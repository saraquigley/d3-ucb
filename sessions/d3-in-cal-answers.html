Now let’s add the chart to Cal Answers

  1. Log into the OBIEE dev-01 environment. If you don't have access to the dev-01 environment, you'll have to just follow along with me for now.
  2. In the shared folder, under the Student Demographics & Outcomes folder, make a local copy the Graduation & Retention Dashboard (using More... Copy and then pasting the folder under My Folders (you can call it somethign like d3-ucb.
  3. Open your local copy of the UGCohort Grad Data query which is used by the Graduation & Retention Dashboard.
  4. On the Cal Answers Analysis Criteria tab, modify the query by changing the Student Headcount Column Properties as follows: in the Column Properties dialogue, on the Data Format tab, uncheck the box that says "Use 1000's Separator"
  5. On the Results tab, modify the Analysis Properties by going to the Data tab and checking the box called "Include Null Values"
  6. On the Results tab, add a Narrative View: Go to New Views... Other View... Narrative View
  7. Here's a handy annotated picture of the Cal Answers Narrative View for your reference.
  8. Add any CSS and HTML text in the Prefix box. It is here that you also instantiate the data array that will hold the output from your query on the Criteria tab.
  9. The Narrative box is where you take the output from your OBIEE query and push it into the data array (called "data"). While you push each row, you can name the columns. Name the columns so that they match the properties that are referenced in the javascript code. For example, the data returned in column 1 should be called "semesterYearLetter".
  10. The Postfix is where most of the javascript code resides, and where the manipulation of the query output happens, as well as where the elements of the visualization are created and appended to the page.
  11. Also, be sure to check the box called "Contains HTML Markup" and if you're returning a lot of rows, the data may be truncated unless you specify this an adequately high number to the "Rows to Display" text box.
  12. Finally, keep in mind that The output will display below the Postfix box. If you see nothing, then something might have gone wrong...
